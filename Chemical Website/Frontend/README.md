# Glide Chemicals Website

![Glide Chemicals](public/logo.svg)

A modern, high-performance frontend website for **Glide Chemicals**, a wholesale chemical distributor. The website focuses on providing a premium, professional experience for B2B clients, featuring an eye-catching design, smooth animations, and a responsive layout.

## ✨ Features

- **Beautiful Splash Screen:** An animated, engaging splash screen that plays every time the site is reloaded, featuring a custom hexagonal flask logo with molecular and liquid animations.
- **Modern UI/UX:** A clean, professional, and responsive interface designed with Tailwind CSS.
- **Dark Mode Support:** A built-in toggle to seamlessly switch between light and dark themes.
- **Dynamic Routing:** Multi-page architecture powered by React Router, including Home, About, Services, Products, and Contact pages.
- **Smooth Animations:** High-quality micro-interactions, page transitions, and floating effects using Framer Motion.
- **Video Backgrounds:** Engaging hero sections using optimized video backgrounds to capture user attention.
- **Custom Branding:** Carefully crafted custom SVG logos and brand colors tailored for the chemical industry.

## 🛠️ Technology Stack

- **Framework:** React.js (via Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Routing:** React Router v6

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/bkrgill/Glide-chemical-website.git
   cd Glide-chemical-website/Frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **View the application**
   Open your browser and navigate to `http://localhost:5173/`
